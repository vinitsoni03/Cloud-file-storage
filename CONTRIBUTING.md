# Contributing

Thank you for your interest in contributing to this project!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/cloud-file-storage.git`
3. Install dependencies: `cd frontend && npm install && cd ../backend && pip install -r requirements.txt`
4. Create a new branch: `git checkout -b feature/your-feature`
5. Make your changes and commit them
6. Push to your branch: `git push origin feature/your-feature`
7. Open a pull request

## Code Style

Frontend:
- Prettier for formatting
- ESLint for linting

Backend:
- Black for code formatting
- isort for import sorting
- Flake8 for linting

## Running Tests

Frontend:
```bash
cd frontend
npm test
```

Backend:
```bash
pytest tests/
```
